# tylerreckart.com
My personal website. There are many like it, but this one is mine. 

### Install
```
git clone https://github.com/tylerreckart/tylerreckart
cd tylerreckart
npm i -g yarn
yarn install
``` 

### Develop
```
yarn develop
```

### Build  
```
yarn build
```
